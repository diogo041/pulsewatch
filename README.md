# PulseWatch

Self-hosted uptime monitoring and incident dashboard deployed on a personal server.

## Goal

Build a full-stack product that monitors websites and APIs, stores uptime history, tracks incidents, and exposes a public status page.

## Planned Stack

- Next.js
- Node.js + TypeScript
- PostgreSQL
- Docker Compose
- GitHub Actions
- Ubuntu VPS

## Planned Features

- monitor creation and management
- scheduled health checks
- response time tracking
- incident detection
- public status page
- alerts
- self-hosted deployment

## Project Structure

- `web/` frontend application
- `api/` backend application
- `docs/` architecture and progress notes
- `infra/` deployment and server configuration
